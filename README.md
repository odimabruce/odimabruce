## About Me

I am a frontend-first software engineer focused on building practical, fast, secure, and user-friendly web applications.

My strongest work is around complex interactive systems: dashboards, workflow builders, drag-and-drop interfaces, calendar tools, e-commerce platforms, API-driven interfaces, and structured data platforms.

I care about clean design, performance, SEO, resource optimization, secure patterns, and software that remains easy to improve over time.

## How I Think About Software

| Principle | What It Means |
|---|---|
| **Useful first** | Build features that solve clear user problems. |
| **Fast by design** | Treat performance as part of the product, not an afterthought. |
| **Clear data flow** | Keep API responses, state, and UI logic easy to reason about. |
| **Secure defaults** | Avoid exposing sensitive data and design safer access patterns. |
| **Reusable UI** | Build components that reduce repeated work and improve consistency. |
| **Simple UX** | Make workflows easier for users to understand and complete. |
